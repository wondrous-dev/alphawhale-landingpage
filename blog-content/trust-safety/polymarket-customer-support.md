## Support on Polymarket

Customer support for crypto-based platforms differs from traditional services. This guide covers how to get help on Polymarket and what to expect.

## Available Support Channels

Polymarket offers several support options:

**Discord community:**
- Active community channels
- Peer support from other users
- Some official staff presence
- Quick responses for common issues

**Documentation:**
- FAQ sections
- How-to guides
- Troubleshooting resources
- Getting started materials

**Social media:**
- Twitter/X presence
- Announcement channels
- Community engagement

**Email support:**
- For specific issues
- Response times vary
- More formal inquiries

## What Support Can Help With

Issues typically addressed:

**Technical problems:**
- Platform access issues
- Transaction problems
- Interface difficulties

**Market questions:**
- Resolution criteria clarification
- Market mechanics
- Trading questions

**Account issues:**
- Connection problems
- Wallet integration

**General inquiries:**
- How things work
- Getting started help

## What Support Cannot Help With

Limitations to understand:

**Reversing blockchain transactions:** Irreversible by design.

**Recovering lost keys:** Only you control your wallet.

**Guaranteeing outcomes:** Trading involves risk.

**Regulatory advice:** Consult appropriate professionals.

**Immediate response:** Response times vary.

## Expectations for Response

What to expect:

**Discord:** Often quick community responses.

**Documentation:** Immediate (self-service).

**Email:** Days potentially, depending on issue.

**Formal issues:** Longer resolution times.

This differs from traditional 24/7 support models.

## Self-Service Resources

Help yourself first:

**FAQ:** Common questions answered.

**Documentation:** Detailed how-to guides.

**Community forums:** Search for similar issues.

**Tutorials:** Getting started resources.

Many issues can be resolved through self-service.

## Effective Support Requests

Get better help by:

**Being specific:** Describe exactly what happened.

**Including details:** Transaction IDs, timestamps, screenshots.

**Checking existing resources:** Demonstrate you've tried self-service.

**Being patient:** Responses take time.

**Being respectful:** Politeness helps.

## Common Issues and Solutions

Frequent problems with answers:

**Can't connect wallet:**
- Try different browser
- Clear cache
- Check wallet compatibility
- Review connection guides

**Transaction pending:**
- Check blockchain explorer
- Wait for confirmation
- Verify gas was sufficient

**Market resolution questions:**
- Check resolution criteria
- Review official sources
- Wait for official resolution

**Position not showing:**
- Refresh interface
- Check correct account
- Verify transaction confirmed

## Community Support

The user community helps:

**Discord channels:** Active discussion.

**Experienced users:** Many willing to help.

**Shared knowledge:** Collective problem-solving.

**Real-time assistance:** Often faster than official channels.

## Alpha Whale Support

For copy trading support:

**Separate from Polymarket:** Alpha Whale has its own support.

**Platform-specific issues:** Contact appropriate platform.

**Combined issues:** May need both platforms' help.

## When to Escalate

Signs you need official support:

- Significant fund issues
- Persistent technical problems
- Dispute situations
- Security concerns
- Issues community can't resolve

## Record Keeping

Help support help you:

**Save transaction details:** IDs, hashes, timestamps.

**Screenshot issues:** Visual evidence helps.

**Document steps:** What you did, what happened.

**Keep communications:** Reference previous exchanges.

## Realistic Expectations

Understanding the model:

**Not traditional support:** Different from banks or brokers.

**Community-driven:** Much help comes from users.

**Self-service emphasis:** Documentation over hand-holding.

**Crypto-native approach:** Assumes some user capability.

## Conclusion

Polymarket support differs from traditional platforms. It emphasizes:
- Self-service documentation
- Community assistance
- Reasonable but not instant official response

For most issues:
1. Check documentation first
2. Search community forums
3. Ask in Discord
4. Contact official support for unresolved issues

Set appropriate expectations for a crypto-based platform. Many issues resolve through self-service and community help. Official support exists but operates differently from traditional customer service models.
